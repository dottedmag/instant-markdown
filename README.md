# Instant Markdown

Emacs mode to instantly render Markdown in browser.

## How to use

- Load
- Use `instant-markdown-mode` in any Markdown file
- Open the URL displayed in minibuffer in browser
- Type in Markdown
- Enjoy

- [See in action](https://youtu.be/nAtD-qqAKWo)
